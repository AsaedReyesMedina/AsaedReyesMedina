# 📦 Mi Librería de Componentes en React con Tailwind CSS 🌟

## 🔍 ¿Qué encontrarás en esta librería?

- **Componentes reactivos**: Todos los componentes están optimizados para aplicaciones de React.
- **Personalización fácil**: Gracias a Tailwind CSS, cada componente es totalmente personalizable para adaptarse a cualquier estilo.
- **Compatibilidad con Storybook**: Visualiza y prueba cada componente en un entorno aislado usando Storybook. 🧑‍💻

---

Sigue leyendo para ver cada componente en acción y descubrir cómo puedes incorporarlos en tus proyectos. 💪 ¡Es hora de construir interfaces increíbles!

## 🚀 Instalación de `@AsaedReyesMedina/react-library` 💻

¡Bienvenido! Aquí te mostramos cómo instalar **@AsaedReyesMedina/react-library** en tu proyecto. Sigue estos sencillos pasos para integrar nuestra librería de componentes y empezar a crear interfaces increíbles. 🎉

### 🔨 Paso 1: Instalar con `npm` o `yarn`

Puedes instalar esta librería usando tu gestor de paquetes favorito. 🚀

#### Opción 1: Usando `npm`

Ejecuta el siguiente comando en tu terminal:

```bash
npm install @AsaedReyesMedina/react-library
```

#### Opción 2: Usando `yarn`

```bash
yarn add @AsaedReyesMedina/react-library
```

### 📦 Paso 2: Importar la Librería en tu Proyecto

Una vez que la instalación esté completa, importa los componentes que desees utilizar en tu proyecto. Aquí tienes un ejemplo de cómo hacerlo:

```bash
import { Button, Card } from '@AsaedReyesMedina/react-library';
```

¡Eso es todo! Ahora puedes utilizar nuestros componentes listos para producción, con el estilo de Tailwind CSS que te encanta. 🥳✨

### 🚩 Nota importante

Si aún no tienes Tailwind CSS configurado en tu proyecto, asegúrate de instalarlo siguiendo [esta guía de instalación de Tailwind CSS](https://tailwindcss.com/docs/installation) para aprovechar todas las capacidades de diseño de nuestros componentes. 📘

Ya estás listo para construir experiencias de usuario únicas y atractivas con @AsaedReyesMedina/react-library. ¡Feliz codificación! 💻❤️
